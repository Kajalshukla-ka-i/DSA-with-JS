        Algorithm complexity : 

        Space complexity:  
            refers to the amount of memory an algorithm or program uses to complete its execution. It measures the total amount of memory required by an algorithm, including both fixed and variable components.
            
            OR

            space complexity is the amount of space required to solve a problem
            space complexity = Auxiliary space + input size

        Time complexity : 
            The time complexity is mainly calculated by counting the number of steps to finish the execution
